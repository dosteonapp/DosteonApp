# Dosteon

Dosteon is an AI-powered dashboard platform supporting role-based interaction for **Restaurants** and **Suppliers**.

## Project Overview

This repository is split into two primary components:

- **[Backend](./backend/README.md)**: Node.js/Express server providing secure API access and admin user management.
- **[Frontend](./frontend/README.md)**: Next.js application delivering the user experience and role-based authentication.

## Quick Start

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/OsteonAI/Dosteon.git
    cd Dosteon
    ```

2.  **Setup Backend**:
    Follow the [Backend Setup Guide](./backend/README.md).

3.  **Setup Frontend**:
    Follow the [Frontend Setup Guide](./frontend/README.md).

## Core Technologies

- **Frontend**: Next.js, TailwindCSS, Formik, Supabase SSR.
- **Backend**: Node.js, Express, Supabase Admin SDK.
- **Database/Auth**: Supabase.
