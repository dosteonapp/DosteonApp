# MVP Roadmap

This document outlines the phased implementation of features for the Dosteon platform.

## Phase 1: Core Vertical Stabilization (MVP v1.0)

_Focus: Replacing all mock data with real API integration for primary workflows._

- [ ] **Inventory Hardening**: Connect log usage and daily stock logs to backend.
- [ ] **Order Tracking**: Implement full order lifecycle (status updates, delivery tracking).
- [ ] **Notifications**: Real-time alerts via Supabase/FastAPI.
- [ ] **Finance Basics**: Real revenue/expense tracking for suppliers.

## Phase 2: Enhanced Features (MVP v1.1)

_Focus: Improving user experience and adding critical business logic._

- [ ] **Onboarding Completion**: Step-by-step wizard for new users.
- [ ] **Advanced Analytics**: Deeper insights for finance and inventory.
- [ ] **Document Management**: Invoices and receipts generation.

## Phase 3: Scaling & Optimization (MVP v2.0)

_Focus: Preparation for larger user base and complex scenarios._

- [ ] **Multi-location Support**: Manage inventory across multiple sites.
- [ ] **Supplier Forecasting**: Predictive ordering suggestions.
- [ ] **Global Search**: Unified search across all domains.
